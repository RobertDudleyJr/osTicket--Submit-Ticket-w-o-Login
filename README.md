<h1>osTicket - Allow Anyone to Submit a Help Desk Ticket</h1>
This tutorial outlines the prerequisites and installation steps for configuring osTicket to allow anyone to submit a help desk ticket without having a login.<br />

<h2>Environments and Technologies Used</h2>

- osTicket (Help Desk Ticketing System)
- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)


<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>List of Prerequisites</h2>

- osTicket must be installed and configured correctly.
